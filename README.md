# Electron Stopwatch

Приложение секундомера на Electron с TypeScript и системным треем.

## Функциональность

- Кнопки управления: Старт, Пауза, Стоп
- Отображение прошедшего времени в окне приложения
- Иконка в системном трее, показывающая время (когда секундомер запущен и не на паузе)
- При закрытии окна приложение сворачивается в трей
- Кнопка "Закрыть" в окне для полного закрытия приложения
- Пункт меню в трее для закрытия приложения

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

или

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Структура проекта

- `src/main.ts` - основной процесс Electron с логикой секундомера и трея
- `src/renderer.ts` - логика UI в renderer процессе
- `src/preload.ts` - мост для безопасной коммуникации между процессами
- `src/index.html` - HTML интерфейс приложения
