# E2E Тесты

E2E тесты для Electron Stopwatch приложения, написанные с использованием Playwright.

## Установка

Перед запуском тестов необходимо установить зависимости:

```bash
npm install
```

Также нужно установить браузеры для Playwright:

```bash
npx playwright install
```

## Запуск тестов

### Все тесты

```bash
npm run test:e2e
```

### С UI режимом

```bash
npm run test:e2e:ui
```

### В headed режиме (с видимым браузером)

```bash
npm run test:e2e:headed
```

## Структура тестов

Тесты находятся в файле `stopwatch.spec.ts` и покрывают следующие сценарии:

- Отображение начального состояния
- Запуск секундомера
- Остановка секундомера
- Пауза и возобновление секундомера
- Отображение времени с миллисекундами
- Отображение статуса секундомера
- Сворачивание и разворачивание окна
- Сохранение состояния при сворачивании

## Примечания

Тесты запускают Electron приложение через webServer в конфигурации Playwright. 
Приложение открывается как локальный файл (`file://` протокол).

Для работы тестов необходимо, чтобы приложение было собрано (`npm run build`).
